# FAQ

## 1. Why is plaintext private key login not supported?

Plaintext private keys are (and should be) a thing of the past, the closest provider would be HiveSigner. Even in web2 passkeys have replaced passwords.

With that said there is a [BeekeeperProvider](https://github.com/aioha-hive/aioha/blob/main/src/providers/custom/beekeeper.ts) ([usage](/docs/core/custom-provider#beekeeper-signer)) but it should only be used for backends.
